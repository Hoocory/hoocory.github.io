Useful Tools

Short summary of tools I have found useful over the years.

The Photon Simulator - PhoSim

PhoSim is a useful tool that can be used in a variety of ways. It can be used to simulate an optical system to extreme detail. Some examples include studying charge diffusion layers in CCDs or simulating optical artifacts such as ghosts. You can also use this more educationally, such as to familiarize yourself with FITS files, showing how telescopes produce files, segment files, and stitching them make your own combined images. PhoSim is completely free and can run on any computer.
